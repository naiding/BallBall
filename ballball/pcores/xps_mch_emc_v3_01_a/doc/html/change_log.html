Changes in v3.00.a, introduced in 11.1

11.1 - Changes in VHDL/Verilog/Netlist sources (.vhd, .v, .ngc, .edn) 



New Features:
  • Support for page mode flash
  • Support for 64 bit external memory
  • Support for 64 bit IPIF native Dwidth
  • Support XCL/XCL2 features using C_MCHx_PROTOCOL parameter
  • Changed the helper libraries
    • proc_common_v2_00_a to proc-common_v3_00_a
    • mch_plbv46_slave_burst_v1_00_a to mch_plbv46_slave_burst_v2_00_a
    • emc_common_v3_00_a to emc_common_v4_00_a
Resolved Issues:
  • Fixed [CR<474993>] : Support for 64 bit memory
  • Fixed [CR<304057>] : Support for page mode flash memory
  • Fixed [CR<470102>] : Fixed the delay in back to back writes
Known Issues / Limitations:
  • Design will fail if Rdclk is not connected properly

11.1 - Changes in tool interface files (.mpd)

  • C_PAGEMODE_FLASH_x paratmeter is added for page mode flash
  • C_MEMx_WIDTH parameter range has been updated to support 64-bit
  • C_MAX_MEM_WIDTH parameter range has been updated to support 64 bit
  • C_MCH_NATIVE_DWIDTH parameter has been updated to support 64 bt IPIF Dwidth
  • C_TPACC_PS_FLASH_x parameter has been added for page read timing
  • Signal names MCH_PLB_Clk and MCH_PLB_Rst has been replaced with MCH_SPLB_Clk and MCH_SPLB_Rst
  • Range of C_SPLB_P2P has been updated [CR <447734>]
  • parameter name C_MCH_PLB_CLK_PERIOD_PS is replaced with C_MCH_SPLB_CLK_PERIOD_PS
  • Updated C_MCHx_PROTOCOL to support DXCL2 writeback option

11.1 - Changes in Tcl script files associated with core (.tcl)

  • Check for "Parameter C_INCLUDE_DATAWIDTH_MATCHING_x should be '1' only when C_MEMx_WIDTH is not equal to C_MCH_NATIVE_DWIDTH"
  • Check for "Parameter C_MCH_NATIVE_DWIDTH can be 64 only when C_NUM_CHANNELS='0'"
  • Check for "Parameter C_SPLB_DWIDTH should be always greater than or equal to C_MCH_NATIVE_DWIDTH for xps_mch_emc core

11.1 - Changes in documentation associated with core

  • Updated DS document for support to page mode flash memories
  • Updated DS document for 64 bit external memory
  • Updated DS document for 64 bit IPIF native Dwidth
  • Updated DS document with page mode and 64 bit memory parameters
  • Updated DS document with XCL & XCL2 support
  • Updated the DS document with timing diagrams
